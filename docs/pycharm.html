Tutorial - pyCharm, Docker, and Python code

pyCharm Prerequisites Connect to git Connect to Docker Use the remote python interpreter

Prerequisites

There are three prerequisites for this tutorial.

  1. You have installed pyCharm.
  2. You have installed Docker. It is a good idea to test your installation: 
    docker run hello-world
  3. You have created a repository on github for python code.

Connect to git

  1. Clone your git repository into pyCharm
  2. Create a src directory in your project in pyCharm.
  3. Goto Settings, expand your Project settings and select Project Structure.
  4. Select the src directory and Mark it as Sources.
  5. Create a gitignore and add .idea. This is an internal pyCharm directory that is not needed in your repository.
  6. Create a python file. Anything that is functional python code will work for this tutorial.

You have now connected pyCharm to git, and can manage your project from within the pyCharm GUI.

Connect to Docker

  1. Make sure Docker is running. If you have not already done so, test your installation. 
    docker run hello-world
  2. Open your project from earlier in pyCharm and goto Settings.
  3. Expand Build, Execution, Deployment and click Docker.
  4. Click the + and select Docker for Windows or whichever is appropriate for your operating system.
  5. Create the Dockerfile and add the code for your project.
  6. At the top of the pyCharm window, click the ADD CONFIGURATION button.
  7. Click the + and select Dockerfile.
  8. If your Dockerfile wasn't automatically added, add it now.
  9. Fill in the name and image tag fields, and make sure Run Built Image is checked.
  10. At the top of the pyCharm window, click the Play button. The Docker console will open and your project will run.

You have successfully connected pyCharm to Docker.

Use the remote python interpreter

  1. At the bottom of the pyCharm window, click <No interpreter> and select Add Interpreter...
  2. Click Docker and select the image that you created in the previous section. Click OK.

You've now successfully added the remote python interpreter from the Docker image to your project.
