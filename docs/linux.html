Working with Linux

Tips & Tricks

You can use clear to clear the terminal if it gets filled up with too many commands.

The tab key is used for autocomplete. For example, type cd Doc and hit tab and the terminal will autocomplete cd Documents.

Ctrl+C can be used to stop a command in terminal safely.

If that doesn't stop the command, Ctrl+Z can be used to force stop it.

You can exit from the terminal by using the exit command.

You can power off or reboot the computer by using the commands sudo halt and sudo reboot.

Login to Docker

docker ps will give you a list of your docker containers.

docker exec -it CONTAINER_NAME /bin/bash will give you a bash shell in the container.

docker exec -it CONTAINER_NAME COMMAND will execute the COMMAND in the container.

Basic Linux Commands

pwd

When you first open the terminal, you are in the home directory of your user. To know which directory you are in, you can use the pwd command. It gives us the absolute path, which means the path that starts from the root. The root is the base of the Linux file system. It is denoted by a forward slash /. The user directory is usually something like /home/username.

ls

Use the ls command to know what files are in the directory you are in. You can see all the hidden files by using the command ls -a.

cd

Use the cd command to go to a directory. For example, if you are in the home directory, and you want to go to the Downloads directory, you type cd Downloads. Remember, this command is case sensitive, so you have to type in the name of the directory exactly as it is.

But what if there's a space? Imagine you have a directory named Raspberry Pi. In this case, when you type cd Raspberry Pi, the shell will take the second argument Pi of the command as a different one, so you will get an error saying that the directory does not exist. Here, you must use an escape character - the backward slash \. The command would look like cd Raspberry\ Pi.

If you just type cd and press enter, it takes you to the home directory. To go up one level to the parent directory, you can type cd ... The two dots represent the parent directory.

mkdir & rmdir

Use the mkdir command to create a directory. For example, if you want to make a directory called DIY, type mkdir DIY. Remember, if you want to create a directory named DIY Hacking, then you need to escape the space character using a backslash \. The command would be mkdir DIY\ Hacking.

Use rmdir to delete a directory. rmdir can only be used to delete an empty directory. To delete a directory containing files, use rm.

rm

Use the rm command to delete files and directories. Use rm -r to delete a directory. This deletes the directory and the files it contains.

touch

The touch command is used to create a file. It can be anything, from an empty txt file to an empty zip file. For example, touch new.txt.

man & COMMAND --help

To learn more about a command and how to use it, use man COMMAND. This shows the manual pages of the command. For example, man cd shows the manual pages of the cd command. Typing in the command name and the argument --help will show how the command can be used.

cp

Use cp command to copy files through the command line. It takes two arguments: The first is the location of the file to be copied, the second is where to copy.

mv

Use mv to move files through the command line. We can also use the mv command to rename a file. For example, if we want to rename the file text to new, we can use mv text new. It takes the two arguments, just like the cp command.

locate

The locate command is used to locate a file in a Linux system, just like the search command in Windows. This command is useful when you don't know where a file is saved or the actual name of the file. Using the -i argument with the command tells it to ignore case. So, if you want a file that has the word hello, it gives the list of all the files in your Linux system containing the word hello when you type in locate -i hello. If you remember two words, you can separate them using an asterisk *. For example, to locate a file containing the words hello and this, you can use the command locate -i *hello*this.
